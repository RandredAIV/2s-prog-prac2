# RU

# Изучение математических основ криптографии

## Описание

Данная работа посвящена изучению математических основ криптографии и реализации ключевых алгоритмов теории чисел, применяемых в современных криптографических системах.

В рамках работы реализованы циклические вычислительные процессы, алгоритмы модулярной арифметики, алгоритм Евклида, методы нахождения обратных элементов по модулю, а также один из криптографических алгоритмов с открытым ключом.

Работа выполнена на двух языках программирования.  
Каждое задание реализовано в отдельном файле и компилируется отдельно.

---

## Структура проекта

Пример структуры файлов:

- Ex1.cpp / Ex1.java — Возведение в степень по модулю  
- Ex2.cpp / Ex2.java — Алгоритм Евклида  
- Ex3.cpp / Ex3.java — Обратный элемент по модулю  
- Ex4.cpp / Ex4.java — Алгоритм шифрования  
- Ex5.cpp / Ex5.java — Цепные дроби и линейные уравнения  
- Ex6.cpp / Ex6.java — Эмуляция атаки  

Каждый файл является самостоятельной программой.

---

## Использование

Каждый файл необходимо компилировать и запускать отдельно.

---

## Сборка и запуск (C++)

Компиляция одного файла:

g++ Ex1.cpp -o ex1  
./ex1  

Для других заданий:

g++ Ex2.cpp -o ex2  
./ex2  

g++ Ex3.cpp -o ex3  
./ex3  

и т.д.

---

## Сборка и запуск (Java)

Компиляция одного файла:

javac Ex1.java  
java Ex1  

Для других заданий:

javac Ex2.java  
java Ex2  

javac Ex3.java  
java Ex3  

и т.д.

# EN

# Study of Mathematical Foundations of Cryptography

## Description

This project focuses on the mathematical foundations of cryptography and implementation of key number theory algorithms used in modern cryptographic systems.

The work includes modular arithmetic algorithms, the Euclidean algorithm, modular inverse computation, and a public-key cryptographic algorithm.

The project is implemented in two programming languages.  
Each task is implemented in a separate file and must be compiled separately.

---

## Project Structure

Example file structure:

- Ex1.cpp / Ex1.java — Modular exponentiation  
- Ex2.cpp / Ex2.java — Euclidean algorithm  
- Ex3.cpp / Ex3.java — Modular inverse  
- Ex4.cpp / Ex4.java — Encryption algorithm  
- Ex5.cpp / Ex5.java — Continued fractions and linear equations  
- Ex6.cpp / Ex6.java — Attack simulation  

Each file is an independent program.

---

## Usage

Each file must be compiled and executed separately.
